Objetivo: Este aplicativo foi desenvolvido para auxiliar no controle de estoque de produtos.

O usuário pode cadastrar produtos informando nome, quantidade e preço. O aplicativo também utiliza o GPS do dispositivo para registrar a localização onde o produto foi cadastrado. 

Funcionalidades:
Cadastrar produtos
Listar produtos cadastrados
Excluir produtos
Capturar localização GPS
Salvar os dados localmente utilizando SQLite
